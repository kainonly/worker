# Weplanx Skeleton

一个简单的 WEB 骨架项目

## 系统环境

| 键值                 | 说明                                                                                                                      |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------- |
| APP_NAME             | 应用名称                                                                                                                  |
| APP_KEY              | 应用密钥                                                                                                                  |
| DB_DSN               | 数据库连接地址,例如 `host=localhost user=gorm password=gorm dbname=gorm port=9920 sslmode=disable TimeZone=Asia/Shanghai` |
| DB_MAX_IDLE_CONNS    | 默认 `5`                                                                                                                  |
| DB_MAX_OPEN_CONNS    | 默认 `10`                                                                                                                 |
| DB_CONN_MAX_LIFETIME | 默认 `3600`                                                                                                               |
| CORS                 | 跨域                                                                                                                      |
