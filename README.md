# Lab Serverless

无服务函数实验项目

## ENVIRONMENT

- **APP_NAME** 应用名称
- **APP_KEY** 应用密钥
- **DB_DSN** 数据库连接地址,例如 `host=localhost user=gorm password=gorm dbname=gorm port=9920 sslmode=disable TimeZone=Asia/Shanghai`
- **DB_MAX_IDLE_CONNS** `5`
- **DB_MAX_OPEN_CONNS** `10`
- **DB_CONN_MAX_LIFETIME** `3600`
- **CORS** Cross-domain
